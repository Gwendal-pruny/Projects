#### Prerequisites
- Ensure you have the following installed:
  - Python 3.x
  - Pygame

#### Installation Steps
1. **Clone the Repository:**
   ```bash
   git clone https://github.com/Gwendal-pruny/PythonPoolGame.git
   cd PythonPoolGame
   ```
2. **Install Dependencies:**
   ```bash
   pip install pygame
   ```
3. **Run the Game:**
   ```bash
   python main.py
   ```
